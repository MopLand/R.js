R.String( string )

执行字符串处理操作。

string 执行字符串处理操作的字符串

.output()

返回字符串自身。

R.String(" example ").output();

.pad( length, string, type )

将字符串使用字符填充至指定长度。

length 目标长度

string 填充内容

type 填充方式，0：从左开始，1：从右 开始，2：两边同时

R.String("R.js").pad( 10, "~", 2 );

.trim()

将字符串两边的空白字符去除。

R.String(" ex ").trim().output() + R.String(" ample ").trim().output();

.length()

返回字符串字节长度。

R.String(" example ").length();

.random( length )

产生一个指定长度的随机字符串，并可以控制它的组成方式（大小写，数字）。

length 字符串长度

upper 是否允许大写字母，默认允许

lower 是否允许小写字母，默认允许

number 是否允许数字，默认允许

R.random(40,true,true,true);

.format()

返回格式化后的字符串。

R.String("今天是{0}年{1}月{2}日").format( R.date("y"), R.date("m"), R.date("d") );

.leftTrim()

将字符串左边的空白字符去除。

R.String(" example ").leftTrim();

.rightTrim()

将字符串右边的空白字符去除。

R.String(" example ").rightTrim();

.stripScript()

将字符串中 Script 标签过滤。

R.String(" example ").stripScript();

.stripTags( [allowed] )

将字符串中 HTML 标签过滤。

R.String( ' example ' ).stripTags();
R.String( '<p>Kevin</p> <br /><b>van</b> <i>Zonneveld</i>' ).stripTags( '<i><b>' );

.encodeHTML()

将字符串中 HTML 标签进行编码。

R.String(" example ").encodeHTML();

.decodeHTML()

将已编码的 HTML 标签进行解码。

R.String(" example ").decodeHTML();

.escapeSymbol()

编码特殊符号。

R.String(" %&+/#=").escapeSymbol();

.toCamelCase()

将 CSS 样式属性名转为 JS 形式。

R.String("z-index").toCamelCase();

.unicode()

将 ASCII 字符串转换为 Unicode。

R.String(" example ").unicode();

.eval()

将字符串进行计算。

R.String("var obj = '123';").eval();

提取 URL 中的指定参数的值。

key 参数名称

url URL 字符串，默认为当前 location.href

R.get("key");
