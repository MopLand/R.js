R.Cookie

执行 Cookies 处理操作。

注意：需要在网络环境中测试，因为读取了 location.domain

.set( key, value )

设置指定 Cookies 项的值。

key cookie对应的键

value cookie对应的值

设置键名称为"key"的cookie值为"123"

 R.Cookie.set( "key" , "123" );

.get( key )

返回指定 Cookies 项的值。

key cookie对应的键

返回键为"key"的cookie的值

R.Cookie.get( "key" );

.unset( key [, path, domain] )

删除指定 Cookies 项和值。

key cookie对应的键

path 可选，cookie创建的路径

domain 可选，默认null，cookie创建的域

删除键名称为"abc"的Cookies 

 R.Cookie.unset("abc");
.clear()

从 Cookies 中移除所有项和值。

 R.Cookie.clear();
