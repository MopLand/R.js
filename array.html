R.Array( array )

执行数组处理操作初始化。

array 执行数组处理操作的数组对象

.output()

返回数据对象

R.Array( ['A','B','C'] ).output().join(",");

.first()

返回数组souce中的第一个元素。

 R.Array( ['A','B','C'] ).first();

.last()

返回数组souce中的最后一个元素。

 R.Array( ['A','B','C'] ).last();

.indexOf( val )

测试在数组中是否包含某个值，如果存在则返回索引值，否则返回 -1。

val 要从数组中查找的值

 R.Array( ['A','B','C'] ).indexOf( 'B' );

.min()

获取数组中最小的元素。

 R.Array( [1,2,3,4,5,6] ).min();

.max()

获取数组中最大的元素。

 R.Array( [1,2,3,4,5,6] ).max();

.clear()

从数组中移除所有元素。

 R.Array( ['A','B','C'] ).clear();
