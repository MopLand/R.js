R.Event( event )

执行交互事件（event）处理操作。

event event 对象

.stop( [type] )

停止事件冒泡和浏览器默认行为。

R.Event( event ).stop();

type 需要停止的内容，1 仅停止冒泡，2 仅停止默认行为，默认为全部停止

.mouse()

获取当前鼠标光标所在位置。

R.Event( event ).mouse();

.element()

获取发生事件的节点对象。

R.Event( event ).element();

.target()

发生当前在处理的事件的节点。

R.Event( event ).target();

.keyboard( code, func )

响应键盘事件。

R.Event( event ).keyboard();

code keyCode 键盘码，-1 为响应全部按键

func 回调函数

func 可接受参数

event 当前 Event 事件

code 当前按键码
